HTML, CSS, SVG design
Before this site, I was only familiar with HTML coding and using tables for a page's layout. Now I'm still no expert, but I can proudly say that I wrote nearly all of this site's custom CSS and HTML. I traded tables for divs and used classes to avoid repeating code. However, the design I came up with was specifically made for a desktop. I had the look I wanted but it didn't translate well to other screen resolutions. But rather than re-write my entire CSS I decided to port my design into the Skeleton responsive framework. In doing so, I learned about media queries and, consequently, managed to retain the same exact look on a desktop while preserving the design language for mobile devices. Moreover I replaced all images with SVG and CSS code, so now I can easily change each section's background color without having to create a corresponding graphic for the "transition triangle." And while my contact information is clearly visible, the code is actually obfuscated to thwart spam-bots and e-mail harvesters.
